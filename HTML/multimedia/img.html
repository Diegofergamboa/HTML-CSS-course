Existen dos tipos de imagenes principales las cuales son lossy y lossless

Las imagenes de tipo lossy son las que tienen pérdida de calidad para adecuarse al formato, estas son principalmente JPG (Para manejar principalmente fotos) y, JPEG (Para formatos de fotos principalmente)

Las imagenes de tipo lossless son las que no tienen pérdida de calidad cuando se está en el proceso de renderización, estas son: PNG8 - Especialmente usado para logos, va rodeado de fondo transparente PNG24 Es para fotografias no tiene limite de pixeles es decir es de más de 256,0,0 SVG (Logos principalmente) Este debería ser por excelencia el tipo de imagenes, ya que el proceso de renderización es excelente para este tipo de formatos
